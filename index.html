HTML, CSS and JavaScript

This is a simple note-taking application that stores all the notes to the browser's local storage. The application is fully responsive.

Weather App

Three-day weather forecast app made with React

This web-app is designed for mobile use but of course, it will also work on a desktop.

The app takes weather data from OpenWeather and the API call is implemented with Axios.

This project helped me to understand the fundamentals of React. The hardest part was to figure out how to get all the weather data with Axios.
